> 🔥 A robust automation testing framework for an Online Exam Portal built using **Java, Selenium WebDriver, and TestNG**.

---

## 📌 Project Overview

This project automates key functionalities of an Online Exam Portal:

- 📝 User Signup & Login  
- 👨‍🎓 Student Dashboard  
- 🧑‍🏫 Admin Panel  
- 📊 Exam Creation & Submission  
- 🔐 Security & Validation Checks

## ⚙️ Setup Instructions

### 🔧 Prerequisites
- Java JDK 8+
- Maven
- Chrome Browser
- ChromeDriver

### 🛠️ Steps

1. Clone repo:
git clone https://github.com/your-username/online-exam-portal.git

2. Install dependencies:
mvn clean install

3. Run tests:
mvn test
